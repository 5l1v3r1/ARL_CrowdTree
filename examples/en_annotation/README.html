ARL CrowdTree example deployment

This document describes how to deploy the example located in the examples/en_annotation/ subdirectory. Prior to proceeding with the steps below, you should have taken the steps described in the SETUP.md file located in the project level directory.

Step 1) Install Apache Ant

This example uses Apache Ant to build the .war file for ARL CrowdTree. If you do not have a copy of Apache Ant installed, please see https://ant.apache.org/

Step 2) Set paths

You will need to set several paths that specify the locations of input and output files. NOTE: Spaces are not permitted in the paths. In examples/en_annotation/annotation_configuration.argmap, update the following lines appropriately. 
 #LogFile specifies the location of the main output log file 
 --LogFile /path/to/use/for/output/arl_crowdtree__log.txt
 #Output specifies the location where dependency trees will be recorded
 ---Output /path/to/use/for/output/arl_crowdtree__parsetrees.conllu
 #DataFiles specifies the file with the sentences to annotate
 ---DataFiles /path/to/installation/examples/en_annotation/en_example_sentences.conllu
In examples/en_annotation/web.xml, update the following line appropriately. 
 <param-value>/path/to/installation/examples/en_annotation/annotation_configuration.argmap</param-value>
In examples/en_annotation/build.xml, update the following line to point to the location of your Apache Tomcat webapps directory. 
 <property name="WEBAPP_DIR" location="/path/to/apache_tomcat/webapps"/>

Step 3) Create and deploy the .war file

From within the /examples/en_annotation directory, type the following command 
 ant deployWarFile
Now start Apache Tomcat. Assuming Apache Tomcat is running locally on port 8080, you can now navigate to the following URL with your internet browser. 
 http://localhost:8080/arlcrowdtree/tree_editor_app.html?tree=0
